Задание: Мы - мебельный магазин. К нам поступают претензии на изделия, нам надо передавать их производителям.

Веб-приложение запускается из файла app.py

Используется JavaScript и AJAX, поэтому работа в Internet Explorer не гарантируется.

/ - страница с анкетой для пользователей

/adminpage - страница дял редактирования и просмотра базы данных

Описание базы данных - в файле db_plan.txt
